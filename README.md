## Descripción
Este proyecto permite gestionar información de países.
El sistema permite agregar, buscar, actualizar, filtrar, ordenar y mostrar estadísticas de países almacenados en un archivo CSV.
====================================================

## Instrucciones de uso

1. Clonar el repositorio
2. Ejecutar el archivo principal:
3. Usar el menú interactivo que se implemento con la libreria questionary para probar las opciones

====================================================

## Funcionalidades

* Agregar países
* Buscar por nombre (parcial o exacto)
* Actualizar población y superficie
* Filtrar por continente y rangos
* Ordenar datos
* Mostrar estadísticas
* salir del menu
====================================================
